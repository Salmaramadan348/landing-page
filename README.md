# 🌞 Soller Landing Page

A modern and responsive landing page built with **HTML5**, **CSS3**, and **JavaScript**. This project recreates a professional UI design with responsive layouts, smooth animations, and an interactive testimonial slider.

## 🚀 Live Demo

👉 https://salmaramadan348.github.io/landing-page/

---

## ✨ Features

- Responsive Design
- Sticky Navigation Bar
- Smooth Scrolling Navigation
- Interactive Card Slider
- Hover Animations
- Decorative UI Elements
- Modern Landing Page Layout
- Mobile, Tablet, and Desktop Support
